# Code of Conduct

## Our Commitment

We want Void to be a respectful, inclusive, and collaborative project.
All contributors, maintainers, and community members are expected to treat each other professionally.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Give actionable technical feedback.
- Accept feedback gracefully.
- Keep discussions focused on the project and facts.

## Unacceptable Behavior

- Harassment, discrimination, or hate speech.
- Personal attacks, insults, or intimidation.
- Trolling, baiting, or deliberately disruptive behavior.
- Publishing private information without consent.
- Any conduct that creates a hostile environment.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and other community interactions related to Void.

## Enforcement

Project maintainers are responsible for interpreting and enforcing this Code of Conduct.
They may remove or edit content, issue warnings, or temporarily/permanently restrict participation for behavior that violates this policy.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers.
Reports will be handled as promptly and confidentially as possible.

## Final Note

We are a student team building this project over the long term. Keep feedback direct, respectful, and helpful so we can all move faster together.
